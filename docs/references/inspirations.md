# Inspirations

## Andrej Karpathy — llm-wiki
This project adopts the idea that knowledge should accumulate in a persistent, structured wiki layer maintained by an LLM, instead of being rediscovered from raw material on every query.

## Jeff Pierce — memory-palace
This project adopts the idea that long-term memory should live outside the model as a durable system layer, capable of routing, persistence, and structured organization across instances or agents.

## Fusion
Z1 Matrix Memory Palace combines:
- Memory Palace for spatial structure and routing
- LLM Wiki for continuous synthesis and reflection
- File-driven command discipline for low-noise multi-agent runtime operation
